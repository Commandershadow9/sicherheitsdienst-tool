# Archivierte Dokumentation

Dieses Verzeichnis enthält veraltete oder abgelöste Dokumentation, die aus historischen Gründen aufbewahrt wird.

## Struktur

### `old-analysis/`
Veraltete Analyse-Dokumente aus der Entwicklungsphase:
- PHASE1_CODE_REVIEW_SUMMARY.md
- PHASE1_TESTING_GUIDE.md
- PR_ANALYSE.md
- PR_INTEGRATION.md

### `old-planning/`
Veraltete Planungsdokumente:
- **Sicherheitstool entwicklung (2).md** - Ursprüngliche Projektplanung (vor MVP)
- **ROADMAP.md** - Historische Roadmap (bis v1.8.0, ersetzt durch TODO.md)

### `old-sessions/`
Veraltete Session-Notizen:
- **SESSION_NOTES.md** - Session-Tracking bis 2025-10-05 (ersetzt durch docs/sessions/)

### `TODO_NEXT_SESSION.md`
Alte TODO-Liste (ersetzt durch docs/TODO.md)

---

## Aktuelle Dokumentation

Für aktuelle Dokumentation siehe:
- `docs/TODO.md` - Aktuelle Aufgaben und Roadmap
- `docs/PROJECT_STRUCTURE.md` - Projekt-Übersicht
- `docs/sessions/` - Aktuelle Session-Notizen
- `docs/planning/` - Aktive Planungsdokumente
- `docs/planning/completed/` - Abgeschlossene Features

---

**Hinweis:** Dateien in diesem Verzeichnis werden NICHT regelmäßig aktualisiert und dienen nur als historische Referenz.
