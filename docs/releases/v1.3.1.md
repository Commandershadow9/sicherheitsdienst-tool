# Release v1.3.1 – Hotfix Absence Decisions

Datum: 2025-10-04

## Highlights

- Absence-Status-Updates korrigieren `decidedById`, sodass der Dev-Compose-Start ohne Fehlermeldung durchläuft.
- Troubleshooting verweist nun explizit auf `SEED_ON_START` für lokale Stacks.

## Release-Checkliste

1. `npm ci && npm run lint && npm run test`
2. `npm run build`
3. `npm version patch --no-git-tag-version`
4. `git commit -am "chore: release v1.3.1"`
5. `git tag v1.3.1`
6. `docker buildx build -t ghcr.io/<org>/sicherheitsdienst-tool:v1.3.1 -t ghcr.io/<org>/sicherheitsdienst-tool:latest --push .`
7. Release Notes aus `CHANGELOG.md` und `docs/releases/v1.3.1.md` ins GitHub-Release übernehmen.

## Deploy-Hinweise

- Keine Datenbankmigration erforderlich.
- Dienste neu durchstarten (`docker compose restart api web`) und `/readyz` bzw. `/system` prüfen.
